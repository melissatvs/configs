# Django

## Começando

- Instalação
`pip install django`

- Criar projeto
`django-admin startproject [nome do projeto] .`

- Criar aplicação
`python manage.py startapp [nome do aplicativo]`

- Criar "super usuário"
`python manage.py createsuperuser`

- Iniciar servidor
`python manage.py runserver`

## Django Rest Framework

- Instalação
`pip install djangorestframework`
